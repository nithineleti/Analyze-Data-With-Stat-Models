# Analyze Data With Statistical Models

This project is a small end-to-end **housing data analysis** using Python and **Statsmodels** in a Jupyter notebook. It covers basic exploratory analysis and building statistical models (e.g., regression) to understand relationships between features and housing prices.

## Contents

- `Statsmodel.ipynb`: main notebook (analysis + modeling)
- `housing.csv`: dataset used by the notebook

## Requirements

- Python 3.9+ (recommended)
- Jupyter Notebook (or JupyterLab)

Python packages used in the notebook:

- `pandas`
- `matplotlib`
- `seaborn`
- `statsmodels`
- `scikit-learn`

## Setup

Create and activate a virtual environment, then install dependencies:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install pandas matplotlib seaborn statsmodels scikit-learn jupyter
```

## Run

Start Jupyter and open the notebook:

```bash
jupyter notebook
```

Then open `Statsmodel.ipynb` and run the cells from top to bottom.

## Notes

- The notebook expects `housing.csv` to be in the same folder as `Statsmodel.ipynb`.
- If you share this repository publicly, confirm the dataset is OK to redistribute.
